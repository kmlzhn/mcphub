MCPHUB — TOOL DIRECTORY FOR AI AGENTS
================================================================================

You are an AI agent. This page lists every tool available on MCPHub.
For each tool you will find: what it does, how to call it, required parameters,
expected output, and a worked example. No human interaction is required.

BASE URL:    http://localhost:3000
AUTH HEADER: X-API-Key: demo-key-123

QUICK START
  1. Pick a tool from the list below.
  2. Send a POST request to  http://localhost:3000/api/tools/{TOOL_ID}/call
  3. Include header  X-API-Key: demo-key-123  and  Content-Type: application/json
  4. Pass required parameters as a JSON body.

DISCOVERY ENDPOINTS (no auth required)
  GET  /api/tools                        — list all tools
  GET  /api/tools?category=finance       — filter by category
  GET  /api/tools/search?q={keyword}     — search by keyword
  GET  /api/tools/{tool-id}              — full schema for one tool
  GET  /llms.txt                         — this content as plain text

RESPONSE ENVELOPE (all tool calls return this shape)
  {
    "tool_id":    string,
    "tool_name":  string,
    "status":     "success" | "error",
    "result":     object,
    "usage": {
      "call_id":     string,
      "duration_ms": number,
      "cost_usd":    number,
      "timestamp":   ISO 8601
    }
  }

ERROR CODES
  401 — missing or invalid X-API-Key header
  400 — missing required parameter  →  response lists which fields are missing
  404 — tool ID not found           →  response suggests GET /api/tools

MCP PROTOCOL (for MCP-native agents)
  SSE endpoint:      http://localhost:3001/sse
  Messages endpoint: http://localhost:3001/messages?sessionId={SESSION_ID}
  MCP tools: search_tools, get_tool_details, call_tool, list_categories, check_balance
