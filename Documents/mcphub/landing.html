How it works

Three steps from idea to income.

No DevOps, no billing code, no sales calls. Deploy once and earn passively.


01
Build your tool
Wrap any capability as an MCP server — a web scraper, a credit API, a translation service, anything with an input and an output. We have SDKs for Node.js, Python, and Go. Most publishers are live within a few hours.
02
Deploy to MCPHub
Run mcphub deploy from your terminal. We host it, version it, and generate machine-readable docs from your JSON schema. AI agents can discover and read your tool's spec without any human involvement.
03
Agents find it. You earn.
Agents browse the catalog, read your docs, and start calling your tool — no human needed. Every call is metered and billed automatically. You keep 85%. Money arrives in your account every Friday.
What we handle

Everything you'd hate building.

The infrastructure that sits between your tool and every AI agent in the world. Built once so you never have to.


Authentication Every agent gets a unique API key. We validate, rotate, and revoke. You never touch auth code.
Rate limiting Per-agent limits, burst protection, and graceful backoff — configurable from your dashboard.
Documentation Auto-generated from your JSON schema. Agents read it in one request and know exactly how to call your tool.
Metering & billing Every call is logged, priced, and billed to the calling agent's balance in real time. Zero billing code on your side.
Weekly payouts Revenue collected and sent to your Stripe or bank account every Friday. No invoices, no chasing anyone.
Analytics Real-time call volume, latency, error rates, revenue, and agent attribution — all in one dashboard.
MCP + REST access Your tool is accessible via REST API and native MCP protocol. Works with Claude, GPT, LangChain, and any custom agent.

Any MCP-compatible agent (Claude, GPT, LangChain bots, custom agents) can connect to any MCP tool without custom integration work.
